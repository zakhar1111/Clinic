1️⃣ Patient books time → booking
    - check if doctor is available
    - add booking to patient
    - notify clinic
    - reserve slot in doctor's schedule
    - notify doctor
    - send confirmation to patient
    - etc.

2️⃣ Clinic creates appointment → appointment
    - link booking to appointment
    - set appointment details (time, doctor, reason)
    - notify patient of appointment creation
    - prepare appointment resources (room, equipment)
    - etc.

3️⃣ Doctor provides service → notes, prescriptions, diagnostics
    - record doctor's notes
    - create prescriptions if needed
    - order diagnostics tests if required
    - update appointment status
    - notify patient of follow-up actions
    - etc.
